# Email-Phone-no-Validator

<h3>1.Resume - Email Validator </h3>
# Install

```
# for windows
$ pip install docx2txt==0.5
$ pip install regex
```

# Output 
```
<email>
<email> is correct
```

<h3>2.Phone - Phone No Validator </h3>
# Install

```
# for windows
$ pip install regex
$ pip install openpyxl
```
# Output:

```
<Phone no>
```
